## üìù Adding Documentation

When adding new documentation:

1. Choose the appropriate category directory
2. Use descriptive filename (UPPERCASE_WITH_UNDERSCORES.md)
3. Update `docs/README.md` and `docs/INDEX.md` if it's a major document
4. Keep root directory clean (only README.md)

See [DOCUMENTATION_MANAGEMENT.md](DOCUMENTATION_MANAGEMENT.md) for detailed guidelines.
